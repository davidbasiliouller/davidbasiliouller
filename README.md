## 🚀 Featured Projects

### 🏭 [Industrial Production Monitoring System](https://github.com/davidbasiliouller/industrial-monitoring-vpn)
> **Thesis project · Grade 9.5/10 · Deployed in production**

End-to-end data pipeline for industrial monitoring: FTP ingestion over VPN → pandas processing → SQLite → real-time JSON dashboard. Built for a concrete production company in SC, Brazil.

`Python` `pandas` `SQLite` `FTP` `VPN` `Chart.js` `Flask`

---

### 🎙️ [Voice-Controlled Servo System](https://github.com/davidbasiliouller/voice-servo-control)
> **Published academic paper · IFC 2025**

IoT system that recognizes Portuguese voice commands offline (Vosk) and drives 3 servo motors via Raspberry Pi over Wi-Fi TCP/IP. Tested at 90–100% accuracy in noisy environments.

`Python` `Vosk` `Raspberry Pi` `pigpio` `PWM` `Socket`

---

### 💧 [IoT Pump Controller](https://github.com/davidbasiliouller/pump-controller)
> **Embedded Systems course**

Flask web app for remote water pump control via Raspberry Pi GPIO relay. Supports manual on/off and time-scheduled activation, accessible from any device on the network.

`Python` `Flask` `Raspberry Pi` `GPIO`
